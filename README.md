# pygame

Velozes e Fuirosos

Bruno Braga, Bruno GT, Igor Dias

Como rodar o jogo: 
Para jogar Velozes e Furiosos, é preciso executar o arquivo main.py e quando a tela inicial aparecer, pressionar 'ENTER' para começar. Use as setas esquerda e direita do teclado ou as teclas 'A' e 'D' para mover o carro entre as faixas, pegue moedas e boosts para dobrar o valor das moedas e aumentar sua pontuação e evite obstáculos e cones na pista, que resultaram em derrota em casos de colisão. A vitória é decretada ao cruzar a linha de chegada. Durante o jogo, pressione 'ESC' para sair a qualquer momento e, nas telas finais, use 'ENTER' para reiniciar a corrida.
